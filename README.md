# PJLink Emulator

Emulates a PJLink projector connected via TCP/IP.


## Documentation

[https://sourceforge.net/p/pjlinkemu/wiki/Home/](https://sourceforge.net/p/pjlinkemu/wiki/Home/)


## HOTKEYS
    
    1	Power State
    2	Input
    3	A/V Mute
    4	Error Code
    5	Lamp Status
    6	Lamp Hours
    
    h	Console Page Down
    j	Console Line Down
    k	Console Line Up
    l	Console Page Up
    
    g	Console Home
    G	Console End
    
    o	Listen
    
    q	Quit Application
    
    :	Command Prompt
    
    
## COMMANDS
    
    port [number]
    
    	Set the TCP/IP port number to listen on.
    
    q | quit | exit
    
    	Quit the application.
